# minitool

轻量的小程序增强框架。

- [x] @mptool/mock: wx API 的简单模拟 (W.I.P)
- [x] @mptool/file: 一套简单的文件与存储 API (Alpha)
- [ ] @mptool/eval: 为小程序添加 eval (Buildng)
- [x] @mptool/enhance: 一套完整的通信框架 (Alpha)

  - 打通任意页面和组件之间的通信
  - 额外的生命周期，支持页面间互相调用进行预加载处理
  - 组件与页面之间的相互引用
  - 支持热更新页面逻辑 (TODO)
